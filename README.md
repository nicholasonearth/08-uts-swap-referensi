# UTS - Fungsi Swap dengan Pass by Reference

## Materi
Konsep *pass by reference* (`&`) di C++ — cara menukar (swap) nilai 2 variabel lewat fungsi tanpa perlu `return`.

## Soal
Tukar nilai 2 variabel (`angka1 = 5`, `angka2 = 10`) menggunakan fungsi `tukar(int &a, int &b)`.

## Status
✅ Sudah lengkap dan berjalan normal, tidak ada bug.

## Cara jalanin
```
g++ main.cpp library.cpp -o main
./main
```
